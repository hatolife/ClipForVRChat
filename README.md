# ClipForVRChat

ClipForVRChat は、VRChat で外部画像URLを使いやすくするための Windows アプリです。

画像を VRChat で扱いやすい 2048x2048px 以内へ縮小し、必要に応じて Discord に投稿して、画像URLをクリップボードへコピーします。

## できること

- 画像を縦横 2048px 以内に縮小
- 複数画像をまとめて処理
- PNG または JPG で保存
- 縮小画像をPCに保存
- Discord Webhook に画像を投稿
- 投稿された画像URLをクリップボードへコピー
- 取得した画像URLを履歴として保存
- 履歴画面からDiscord上の投稿画像を削除

## 使い方

### 1. ダウンロード

最新リリースから `ClipForVRChat-vX.Y.Z-windows-amd64.zip` をダウンロードします。

https://github.com/hatolife/ClipForVRChat/releases/tag/latest

zipファイルを展開し、中にある `ClipForVRChat.exe` を起動してください。

### 2. 初回設定

初めて起動したときは設定画面が開きます。

まずはそのまま保存しても使えます。Discordへ投稿したい場合は、Discord Webhook URL を設定してください。

設定は `ClipForVRChat.exe` と同じ場所の `config.json` に保存されます。通常はこのファイルを直接編集する必要はありません。

### 3. 画像を処理する

アプリ画面に画像ファイルをドラッグ&ドロップします。

複数の画像をまとめてドロップすると、一括で処理できます。処理中は結果欄に進捗が表示され、完了した画像からサムネイルへ切り替わります。

クリップボードにコピーした画像を処理したい場合は、「クリップボード画像を処理」ボタンを押してください。

### 4. URLをコピーする

Discord投稿がONの場合、投稿された画像URLを取得できます。

1枚だけ処理した場合は、設定がONなら自動でクリップボードへコピーされます。結果欄のサムネイルをクリックすると、その画像URLをもう一度コピーできます。

### 5. 結果をクリアする

結果欄右上の「クリア」ボタンで、画面上の結果を消せます。

クリアしてもURL履歴は削除されません。あとから履歴画面で確認できます。

### 6. 履歴を見る

「クリア」ボタンを3秒長押しすると、画像履歴画面を開けます。

履歴画面では、過去に取得したURLの画像を確認できます。Ctrlキーを押しながらクリックすると複数選択、Shiftキーを押しながらクリックすると範囲選択できます。

選択した画像はDiscord上の投稿だけを削除できます。履歴や元画像ファイルは削除されません。

## Discord設定

Discord投稿を使うには、投稿先チャンネルの Webhook URL が必要です。

### Webhook URL の作り方

1. Discordで投稿先にしたいサーバーとチャンネルを開きます。
2. チャンネルの設定を開きます。
3. 「連携サービス」または「インテグレーション」から Webhook を作成します。
4. 作成した Webhook のURLをコピーします。
5. ClipForVRChat の設定画面を開き、「Discord Webhook URL」に貼り付けます。
6. 「Discord投稿」をONにして保存します。

Discord公式の説明:

https://support.discord.com/hc/ja/articles/228383668-%E3%82%A6%E3%82%A7%E3%83%96%E3%83%95%E3%83%83%E3%82%AF%E3%81%AE%E3%81%94%E7%B4%B9%E4%BB%8B#:~:text=Webhook%E3%81%AE%E4%BD%9C%E6%88%90

Webhook URL は、投稿できる権限を持つURLです。知らない人に共有しないでください。

## 設定

設定画面は、画面右上の「設定」ボタンから開けます。

### 出力

- ローカル保存: 縮小画像をPCにも保存します。
- Discord投稿: 縮小画像をDiscordへ投稿し、URLを取得します。
- 1枚時にURLを自動コピー: 1枚だけ処理したときにURLを自動でコピーします。
- 出力形式: PNGまたはJPGを選べます。
- JPEG品質: JPG出力時だけ使います。
- UI表示: 処理後に画面を表示する条件です。通常は `auto` のままで問題ありません。

### 保存

- 出力先フォルダ: 保存先です。初期値は `./output` です。これは `ClipForVRChat.exe` と同じ場所に作られる `output` フォルダを意味します。
- サフィックス: 保存ファイル名の末尾に付ける文字です。例: `image.png` は `image_2048.png` のように保存されます。

### Discord

- Webhook URL: Discordで作成したWebhook URLを貼り付けます。

## 画面

### 初期画面

画像をドラッグ&ドロップする入口です。

左側にドロップ領域、右側に結果欄があります。画像をまだ処理していない場合、結果欄には「まだ処理結果はありません。」と表示されます。

### 結果画面

処理した画像のサムネイルを表示します。

サムネイルにマウスを重ねると、URLコピーの案内が表示されます。サムネイルをクリックすると画像URLをコピーできます。

### 設定画面

保存先、出力形式、Discord投稿などを変更できます。

出力先フォルダは、入力欄へ貼り付けても、「選択」ボタンからフォルダを選んでも設定できます。

### 使い方画面

アプリ内で基本的な使い方を確認できます。

Discord Webhook の作り方を確認するための公式リンクもあります。

### 情報画面

アプリのバージョン、ライセンス、GitHub、バグ報告先を確認できます。

バグ報告:

https://github.com/hatolife/ClipForVRChat/issues

### 画像履歴画面

過去に取得した画像URLを確認できます。

クリア済みやDiscord削除済みの画像も含めて表示されます。「削除済みURLの履歴を削除」を押すと、すでに参照できないURLの履歴だけを整理できます。

## 保存されるファイル

- `config.json`: アプリ設定
- `history.json`: 取得した画像URLの履歴
- `output` フォルダ: ローカル保存した縮小画像
