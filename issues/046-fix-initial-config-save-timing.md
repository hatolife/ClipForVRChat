# 初回設定を保存せず終了した場合に次回も初期設定画面を出す

## 問題

v0.1.6 で、初回起動時に設定画面を開くだけで `config.json` が作成される。ユーザーが保存せずに終了しても設定ファイルが残るため、次回起動時に初期設定画面にならない。

## 期待する挙動

初回起動で設定画面を開いただけでは `config.json` を作成しない。ユーザーが保存した場合にだけ設定ファイルを作成する。

## 受け入れ条件

- `config.json` がない状態で設定を読み込んでも、ファイルが作成されない。
- 初回設定画面で保存せず終了した場合、次回起動時も初期設定画面になる。
- 設定を保存した場合は、従来どおり `config.json` が作成される。
