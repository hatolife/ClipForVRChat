# Discord公式ヘルプリンクが開けない

## 問題

使い方画面の「Discord公式: ウェブフックのご紹介」リンクが環境によって開けない。

## 原因候補

URLに `#:~:text=` のテキストフラグメントが含まれており、Wails の外部ブラウザ起動や Windows 側のURL処理で失敗している可能性がある。

## 対応内容

- アプリ内の Discord Webhook 公式ヘルプ URL から `#:~:text=` を削除する
- README の同リンクも通常の記事URLへ統一する

## 完了条件

- 使い方画面と設定画面の Discord 公式リンクが通常の記事URLを開く
- README のURLも同じ通常記事URLになっている
