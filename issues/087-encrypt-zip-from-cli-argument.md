# 087 zipファイル引数を公開鍵で暗号化する

## 問題

不具合報告用データを手動で用意した場合、アプリから公開鍵暗号化だけを実行する手段がない。

## 期待する挙動

- `ClipForVRChat.exe path\to\data.zip` のようにzipファイルを1つ渡して起動すると、UIを起動せず公開鍵で暗号化する。
- 出力ファイルは入力zipと同じ場所に `<元ファイル名>.gpg` として作成する。
- 暗号化は診断データと同じ `poppo@hato.life` 公開鍵を使い、binary literal dataとして扱う。
- `--version` / `--help` は従来通り動作する。

## 受け入れ条件

- zip単体引数で `.zip.gpg` が生成される。
- 暗号化後ファイルにzip内ファイル名が平文で含まれない。
- zip以外の位置引数は従来通り画像処理へ回る。
