# 062 issue一覧の対応バージョン列名を整理する

## 状態

完了

## 問題

`issues/README.md` の列名が `対応バージョン/コミット` のままだが、リリース時に対応バージョンを記録する運用へ寄せたため、列名と内容が合っていない。

## 期待する挙動

`issues/README.md` の列名からコミット表記を外し、対応バージョンを記録する列であることが分かる。

## 受け入れ条件

- `issues/README.md` の表ヘッダが `対応バージョン` になっている。
- 新しい整理チケットが `issues/README.md` に掲載されている。
