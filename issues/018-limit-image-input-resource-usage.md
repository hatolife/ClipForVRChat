# 画像入力のサイズ上限とピクセル数上限

## 問題

画像ファイルを丸ごと読み込んでからデコードしており、ファイルサイズやデコード後のピクセル数に上限がない。巨大画像や細工画像によりメモリ・CPUを過剰に消費する可能性がある。

## 期待する挙動

処理前にファイルサイズと画像寸法を検証し、上限を超える画像は分かりやすいエラーで拒否する。

## 受け入れ条件

- ファイル入力は最大ファイルサイズを超えたら処理しない。
- クリップボード入力も最大バイト数を超えたら処理しない。
- `image.DecodeConfig` でピクセル数上限を確認してからデコードする。
- 上限超過時の単体テストがある。
