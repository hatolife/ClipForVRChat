# ローカルWindowsビルドのバージョンを指定できるようにする

## 問題

`scripts/build-windows-from-wsl.sh` は最新ローカルタグを自動採用するため、リリース候補確認時に `v0.1.6` タグ未作成または削除済みだと、exeの表示バージョンが旧タグになる。

## 期待する挙動

ローカルWindowsビルド時に、確認したいリリースバージョンを明示できる。

## 受け入れ条件

- 環境変数または引数でバージョンを上書きできる。
- 既存のタグ自動検出挙動は維持される。
- `--version` 表示とWindowsプロパティに指定バージョンが反映される。
