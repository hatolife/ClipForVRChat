# 061 リリース時にissue一覧へ対応バージョンを記録する

## 状態

完了

## 問題

リリース済みの作業でも `issues/README.md` の対応バージョンがコミットのみ、または `未リリース` のまま残ると、どのリリースに含まれた変更か後から確認しづらい。

## 期待する挙動

リリース作業の一部として、対象チケットの対応バージョンを `issues/README.md` に記録する。
既存の完了チケットも、タグから分かる範囲で対応バージョンを補完する。

## 受け入れ条件

- `AGENTS.md` に、リリース時は `issues/README.md` の対応バージョンを更新するルールがある。
- 既存の完了チケットに、分かる範囲で対応バージョンが記載されている。
- 新しい整理チケットが `issues/README.md` に掲載されている。
