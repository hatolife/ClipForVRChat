# 044 v0.1.6 Release workflow のVersionInfo生成失敗修正

## 状態

完了

## 問題

v0.1.6タグのRelease workflowで、`src/build/windows/info.json` の親ディレクトリが存在せず、Windows VersionInfo生成ステップが失敗した。

## 期待する挙動

クリーンチェックアウトのGitHub Actions環境でも、Windows VersionInfo用の `info.json` を生成してReleaseビルドが成功する。

## 受け入れ条件

- `scripts/write-wails-windows-info.mjs` が出力先の親ディレクトリを自動作成する。
- WSLローカルビルドスクリプトが、`src/build/windows/info.json` が未作成の状態でも動作する。
- v0.1.6のRelease workflowが成功する。
