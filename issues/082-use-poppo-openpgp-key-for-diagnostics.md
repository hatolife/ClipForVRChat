# 082 診断パッケージ暗号化鍵をpoppo@hato.lifeへ変更する

## 問題

診断パッケージ暗号化用に指定していた `release-signing@hato.life` の公開鍵は署名専用で、暗号化用途に利用できない。

## 期待する挙動

- 診断パッケージ暗号化には `poppo@hato.life` の公開鍵を使用する。
- 公開鍵に暗号化用サブキーがある場合、診断パッケージを作成できる。
- 作成されたファイルから、復号なしにzip内ファイル名や内容が分からない。

## 受け入れ条件

- 埋め込み公開鍵が `poppo@hato.life` の鍵に差し替わっている。
- 診断パッケージ作成テストが成功ケースを確認している。
- 暗号化後ファイルに `config.json` や `history.json` などのzip内ファイル名が平文で含まれない。
