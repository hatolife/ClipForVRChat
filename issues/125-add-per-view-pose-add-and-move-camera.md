# 構図ごとに現在Pose追加とカメラ移動ボタンを配置する

## 問題

`現在Poseから追加` が構図一覧のヘッダーにあり、どの構図操作なのか分かりにくい。
また、設定済みPoseへVRChatカメラを移動させる操作がなく、数値や保存済み構図をゲーム内で確認しづらい。

## 期待する挙動

- 各構図カード内から現在Poseを新しい構図として追加できる。
- 各構図カード内から、その構図の設定値へVRChat User Cameraを移動できる。
- カメラ移動は撮影やDiscord投稿を行わず、OSCでPoseと構図オプションだけを適用する。

## 受け入れ条件

- `現在Poseから追加` ボタンが各構図カード内にある。
- `このPoseへカメラ移動` ボタンが各構図カード内にある。
- `このPoseへカメラ移動` は `/usercamera/Mode` と `/usercamera/Pose`、Zoom/表示マスク等の構図オプションを送信する。
- 構図が見つからない場合やOSC送信失敗時は画面にエラーを出す。
- 既存の現在Pose保存、テスト撮影、自動撮影を壊さない。

## 対応内容

- 構図一覧ヘッダーの `現在Poseから追加` を外し、各構図カード内へ配置した。
- 各構図カードに `このPoseへカメラ移動` を追加した。
- `MoveCameraToView` APIを追加し、撮影せずに `/usercamera/Mode`、`/usercamera/Pose`、Zoom/表示マスク等を送信するようにした。
