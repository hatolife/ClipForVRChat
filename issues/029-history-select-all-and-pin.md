# 画像履歴の全選択とピン止め

## 背景

画像履歴画面で大量の履歴を整理するとき、全選択操作と削除対象から外す保護操作がない。

## 対応内容

- Ctrl+A で履歴を全選択できるようにする
- 履歴画面上部に全選択ボタンを追加する
- 各履歴カード右上にピンボタンを追加する
- ピン済み履歴は色を変えて表示する
- ピン済み履歴は Discord 削除と削除済みURL履歴削除の対象外にする

## 完了条件

- Ctrl+A と全選択ボタンで履歴が全選択される
- 各画像カードの右上にピンボタンが表示される
- ピン状態が履歴ファイルに保存される
- ピン済み画像は一括削除系操作で削除対象にならない
