# 使い方画面とWebhook発行案内

## 問題

アプリ内に使い方を丁寧に確認できる画面がない。Discord Webhook URL の発行方法も、ユーザーが設定前に参照しづらい。

## 期待する挙動

- ヘッダーに「使い方」ボタンを追加する。
- 使い方画面を追加し、基本的な利用手順を丁寧に説明する。
- 使い方画面では以下を説明する。
  - 画像ファイルをドラッグ&ドロップして処理できること。
  - 複数画像をまとめて処理できること。
  - クリップボード画像を処理できること。
  - 設定画面でローカル保存、Discord投稿、出力形式、出力先などを変更できること。
  - 結果サムネイルをクリックするとURLをコピーできること。
  - Discord投稿にはWebhook URLが必要なこと。
- Discord公式のWebhook説明ページへのリンクを表示し、クリックでブラウザを開く。

## 受け入れ条件

- ヘッダーの「使い方」ボタンから使い方画面を開ける。
- 使い方画面から通常画面へ戻れる。
- Webhook URL発行方法のリンクがブラウザで開く。
- 使い方画面の文章が日本語で、初めて使うユーザーにも次の操作が分かる。
